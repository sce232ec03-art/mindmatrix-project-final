Namma-Hasiru 🌿
*Android App for Natural Resources using Generative AI*

About
Namma-Hasiru is an Android application that helps users learn about and manage natural resources in their local area. The app uses Generative AI to provide insights, recommendations, and real-time information related to water, soil, forests, and other natural assets.  
"Namma-Hasiru" means "Our Green" in Kannada, reflecting the focus on sustainability and local environmental awareness.

Features
- *AI-Powered Insights*: Get AI-generated explanations and advice about natural resources.
- *Resource Mapping*: View nearby water bodies, forests, and soil types.
- *Sustainability Tips*: Receive personalized suggestions to conserve resources.
- *Offline Access*: Access basic resource data without internet.
- *Multilingual Support*: Designed with local users in mind.
